System Overview

This system continuously monitors global news sources, applies machine learning classification to identify crisis events, extracts geographic entities using natural language processing, and generates interactive maps for humanitarian analysis.

The pipeline processes articles from the GDELT Project, uses transformer models for zero-shot crisis classification, and employs spaCy for named entity recognition and geocoding.

Technology stack: Python, Transformers, spaCy, GDELT API, Folium, GitHub Actions
