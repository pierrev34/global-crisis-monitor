System Overview

ARGUS monitors 15+ trusted sources (NGOs, human rights orgs, disaster monitors, and regional media) and updates daily via CI. It applies stricter, rule-based classification to avoid soft-news false positives, extracts locations with NLP + geocoding, and renders an interactive crisis map with keyword search, category filters, heatmaps, and data window notes.

Sources include Human Rights Watch, Amnesty International, UN OCHA, ReliefWeb, Radio Free Asia, USGS/GDACS, and others. Location extraction uses spaCy + Nominatim with crisis-zone overrides (e.g., Gaza, Xinjiang). Search supports aliases (e.g., USA/US/U.S., DRC, Gaza).

Technology stack: Python, spaCy NER, Geopy, Folium, NGO RSS feeds, GitHub Actions
